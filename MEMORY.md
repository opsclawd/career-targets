# Workspace Memory

## Mission

Build and maintain a Calgary-focused, direct-employer, software-engineering job pipeline.

## Non-negotiables

- Calgary and Alberta relevance first.
- Canada-remote is acceptable only when clearly valid for a Calgary-based candidate.
- Direct employer career pages beat aggregators.
- The registries are the durable truth, not session memory.
- Net-new findings matter more than re-reporting old inventory.

## Intended outputs

- a growing employer registry
- a clean jobs ledger
- short daily diffs
- weekly review notes that improve future sourcing
